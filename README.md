# one-plus-one

 A simple maths game built with vuejs inspired by the android games of the sort. 

 ## [Try it now](https://oneplusoneeazy.herokuapp.com/#/)

----

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
